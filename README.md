# Система автоматизации кадрового учета

Десктопное приложение на Avalonia UI (.NET 8) для управления кадрами в небольшой фирме.

## Технологии
- Avalonia UI 11.3.9
- .NET 8.0
- PostgreSQL + Entity Framework Core
- MVC архитектура

## Функционал

### Модели данных (8 таблиц):
1. **Employee** - сотрудники
2. **Department** - отделы
3. **Position** - должности
4. **Vacation** - отпуска и больничные
5. **Contract** - трудовые договоры
6. **Salary** - начисление зарплаты
7. **WorkSchedule** - табель учета рабочего времени
8. **Document** - документы сотрудников

### Интерфейс:
- Список сотрудников (поиск, фильтрация, CRUD)
- Карточка сотрудника (полная информация + вкладки)
- Справочник отделов
- Справочник должностей
- Учет отпусков (с согласованием)
- Начисление зарплаты
- Табель рабочего времени
- Отчеты (по сотрудникам, отделам, зарплатам, отпускам, дни рождения)

## Запуск

1. Настроить PostgreSQL (см. DATABASE_SETUP.md)
2. Создать базу `hr_automation` в pgAdmin4
3. Обновить пароль в `appsettings.json`
4. Применить миграции: `dotnet ef database update`
5. Запустить: `dotnet run`

## Начальные данные
При первом запуске автоматически создаются:
- 4 отдела (Администрация, Бухгалтерия, IT, Продажи)
- 5 должностей (Директор, Главбух, Программист, Менеджер, Бухгалтер)
